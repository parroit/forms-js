# forms 

2way form binding

[![Build Status](https://secure.travis-ci.org/parroit/forms.png?branch=master)](http://travis-ci.org/parroit/forms) [![NPM version](https://badge-me.herokuapp.com/api/npm/forms.png)](http://badges.enytc.com/for/npm/forms) 

## Getting Started
Install the module with: `npm install forms`

```javascript
var forms = require('forms');
forms.awesome(); // "hello "
```
